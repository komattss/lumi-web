# Quick Start Guide

## Step 1: Install Dependencies

Open a terminal in the project directory and run:

```bash
npm install
```

## Step 2: Start Development Server

```bash
npm run dev
```

## Step 3: Open in Browser

Navigate to: **http://localhost:3000**
